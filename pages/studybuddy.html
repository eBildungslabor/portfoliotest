WIP: StudyBuddy

A fully-responsive application designed to help users study through the use of flashcards.

This application is a work-in-progress begun by Abdi Abdikarin, Maggie Alberts, Chetan Audipudy, Chris Banaszak, and Kara Mui.

The final design is intended to allow users to study with sets of flashcards (premade or their own creation), complete practice tests, track their progress during review sessions, and challenge themselves with extended learning opportunities.

Instructions

You can play with StudyBuddy on Heroku.

You will need to have the following Node.js packages installed prior to running this application locally:

  • babel-cli
  • babel-core
  • babel-loader
  • babel-plugin-transform-class-properties
  • babel-preset-env
  • babel-preset-react
  • bcrypt
  • body-parser
  • classnames
  • css-loader
  • express
  • fs
  • live-server
  • mysql2
  • node
  • nodemon
  • path
  • react
  • react-dom
  • react-modal
  • react-router-dom
  • reload
  • sequelize
  • style-loader
  • validator
  • webpack
  • webpack-dev-server

To install all packages quickly, run navigate to StudyBuddy's directory and run "npm install" in the command line.

For the production build, run "npm run build:prod" and "npm run start." For the development build, run "npm run dev."
