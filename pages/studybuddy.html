StudyBuddy

StudyBuddy is a fully-responsive application designed to help users study. Users can create their own sets of flashcards and use them to review important concepts.

This application was created by Abdi Abdikarin, Maggie Alberts, Chetan Audipudy, Chris Banaszak, and Kara Mui.

Instructions

You can play with StudyBuddy on Heroku.

Prior to running this application locally, navigate to StudyBuddy's directory and run "yarn install" in the command line. Repeat this step in the "client" folder. Then return to the root directory and run "yarn start" to deploy the application.
